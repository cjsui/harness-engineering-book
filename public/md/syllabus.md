# Harness Engineering 課程綱要

## 課程定位

本課程以真實 agent harness 作為主案例，帶學生從概念理解、系統拆解，到最終以 Python 重建 `mini harness`。它同時可作為大學課程、研究導向讀書會，或工程師自主學習教材。

## 目標學生

具備基本程式能力，熟悉 Python / CLI / Git，想從 AI coding 的使用者提升為能理解與設計 agent harness 的工程學習者。

## 學習成果

完成本課程後，學生應能看懂 `claw-code` 的核心架構、清楚說明 runtime / tools / permissions / session / testing 的互動關係，並獨立規劃與實作一個最小可行的 `Python mini harness`。
